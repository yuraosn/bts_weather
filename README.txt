INTRODUCTION
------------

This module was created for demonstration purposes to display weather in the block.
Weather is taken from a third-party site: bulk.openweathermap.org
Cities available for selection are taken from the archive of the English version:
   http://bulk.openweathermap.org/sample/
For testing, you can use this token:
   eab0aae7fdfa2caf0aee0bf062bf1860


RECOMMENDED MODULES
-------------------

 * Smart IP (https://www.drupal.org/project/smart_ip):
   After switching on the specified module, additional settings will appear in the administrative page of the module,
   and a button for determining the location will appear in the weather display unit (it does not work on the local version).
